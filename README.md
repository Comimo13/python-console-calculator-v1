# Python Console Calculator v1

A simple command-line calculator built with Python.

## 🚀 Features

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Floor division (//)
- Power (**)
- Input error handling (ValueError)
- Division by zero protection

---

## 🧠 How it works

The program:
1. Asks user to choose an operation
2. Takes two numbers as input
3. Performs calculation using functions
4. Prints the result

---

## 🛠️ Technologies

- Python 3
- Basic functions
- Try/except error handling
